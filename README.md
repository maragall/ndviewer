# NDViewer HCS
High-Content Screening viewer with plate overview and FOV navigation

## Setup

1. **Install environment:**
```bash
chmod +x setup.sh
./setup.sh
```

2. **Create desktop shortcut (optional):**
```bash
python3 utils/create_desktop_shortcut.py
```

## Launch

```bash
conda activate ndv
python -m ndviewer_hcs.ndviewer_hcs
```

Or use the desktop shortcut: **NDViewer HCS**
